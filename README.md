# 세번째미술관

## 리액트 설정

### npm install 하는 방법

저장소 복제 후 vscode 터미널 - git bash에서 다음 명령어를 실행합니다.

경로이동

    cd thirdmuseum/tm-front

해당폴더로 이동이 제대로 됐다면 설치 후 실행 확인

    npm install
    npm start

### 패키지 설치

##### 부트스트랩 설치 

    npm install bootstrap@5.2.3
  
##### router dom 설치

    npm install react-router-dom@5.3.4
  
##### axios 설치

    npm install axios@1.3.3
